Search for and compile today's news from Uttar Pradesh, India using web browser automation.

TASK:
1. Open a Chrome browser using Playwright
2. Search for "Uttar Pradesh news today" on Google
3. Visit at least 4-5 major Indian news sources (e.g., Times of India Uttar Pradesh, Hindustan Times, Indian Express, NDTV, The Hindu, etc.)
4. Extract the top news stories from today including:
   - Headlines
   - Brief summaries (2-3 sentences per story)
   - Source and timestamp if available
   - Key details (who, what, where, when)
5. Create a comprehensive markdown report with:
   - Title: "Uttar Pradesh News - [Today's Date]"
   - Organized by category if possible (politics, crime, development, etc.)
   - At least 10-15 distinct news stories
   - Source attribution for each story
6. Save the report to logs/uttar-pradesh-news-today.md

REQUIREMENTS:
- Use Playwright with headless Chrome
- Sort stories by relevance and recency
- Include only today's news (check timestamps/dates)
- Provide diverse coverage across topics
- Ensure all information is properly attributed
